# 🧠 Интеллектуальная викторина «Что? Где? Когда?»

Консольная игра в формате легендарной телевизионной викторины с ограничением по времени, проверкой знаний и системой начисления очков.

## ✨ Особенности

- ⏱️ **Таймер с визуальным обратным отсчётом** на каждый вопрос
- 🔍 **Умная проверка ответов** с поддержкой синонимов и альтернативных формулировок
- 💡 **Подробные пояснения** к каждому вопросу
- 🎯 **Система очков** — сложность вопроса влияет на количество баллов
- 📊 **Итоговая статистика** с персональной оценкой
- 🌐 **Корректная работа с русским языком** (кодировка CP1251)
- 📁 **Загрузка вопросов из JSON** — легко расширять базу

## 🎮 Как играть

1. На каждый вопрос даётся ограниченное время
2. Введите ответ текстом и нажмите **Enter**
3. Если не успели — вопрос засчитывается как ошибка
4. После ответа (или таймаута) вы увидите:
   - Правильный ответ
   - Все принимаемые альтернативы
   - Категорию и сложность вопроса
   - Историческое/научное пояснение
5. Игра автоматически переходит к следующему вопросу

- Windows (используются WinAPI функции)

🔧 Технические детали
Кодировка
•	Исходники (.cpp/.h): CP1251
•	JSON-файл: UTF-8 (автоматически конвертируется в CP1251 при загрузке)
•	Консоль: CP1251 для корректного отображения русского текста
Таймер
Реализован через ReadConsoleInput с визуальным обратным отсчётом в реальном времени. При истечении времени ввод принудительно прерывается.
Проверка ответов
•	Нечувствительность к регистру
•	Игнорирование начальных и конечных пробелов
•	Поддержка альтернативных формулировок из JSON
•	Корректная обработка русских букв в кодировке CP1251
Система очков
•	Баллы начисляются пропорционально сложности вопроса (1–10)
•	Максимальный счёт зависит от набора вопросов
📊 Итоговая статистика
После прохождения всех вопросов выводится:
•	Количество правильных ответов / всего вопросов
•	Процент правильных ответов
•	Набранные баллы / максимально возможные
•	Персональная оценка (от «В следующий раз всё получится» до «ИДЕАЛЬНО! Вы гений!»)
