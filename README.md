# 🧠 Интеллектуальная викторина «Что? Где? Когда?»

Консольная игра в формате легендарной телевизионной викторины с ограничением по времени, проверкой знаний и визуальным сопровождением.

## ✨ Особенности

- ⏱️ **Таймер 30 секунд** на каждый вопрос
- 🖼️ **Автоматический показ картинок** после ответа
- 🔍 **Умная проверка ответов** (учёт синонимов и альтернативных формулировок)
- 💡 **Подробные пояснения** к каждому вопросу
- 🎯 **Подсчёт очков** и итоговая статистика
- 🌐 **Поддержка русского языка** (кодировка Windows-1251/CP866)
- 🚫 **Автоматическое закрытие картинок** после просмотра
- 📊 **Сохранение результатов** в файл

## 🎮 Как играть

1. На каждый вопрос даётся **30 секунд**
2. Введите ответ текстом и нажмите Enter
3. Если не успели — вопрос засчитывается как ошибка
4. После ответа (или таймаута) вы увидите:
   - Правильный ответ
   - Историческое/научное пояснение
   - Иллюстрацию (откроется автоматически)
5. Картинка показывается 5 секунд, затем закрывается
6. Игра автоматически переходит к следующему вопросу

🔧 Технические детали
Кодировка
Вывод: Windows-1251 (SetConsoleOutputCP(1251))

Ввод: CP866 (SetConsoleCP(866))

Введённый текст автоматически конвертируется для корректного сравнения

Таймер
Реализован с использованием отдельного потока (std::thread). При истечении времени поток ввода отсоединяется (detach), чтобы не блокировать выполнение программы.

Проверка ответов
Поддерживает:

Разные регистры

Начальные и конечные пробелы

Альтернативные формулировки (например, «да винчи» вместо «леонардо да винчи»)

Русские буквы (с учётом кодировки)

📊 Итоговая статистика
После прохождения всех вопросов выводится:

Количество правильных ответов

Процент правильных ответов

Персональная оценка (от «В следующий раз всё получится» до «ИДЕАЛЬНО! Вы гений!»)

Результаты также сохраняются в файл quiz_results.txt.

🛠️ Возможные проблемы и решения
Проблема	Решение
Не открываются картинки	Проверьте, что файлы изображений лежат в той же папке, что и .exe
Неправильно отображается русский текст	Убедитесь, что файлы сохранены в кодировке Windows-1251
Таймер не срабатывает	Убедитесь, что компилятор поддерживает C++11 и подключена библиотека <thread>
Ошибка taskkill при закрытии картинок	Игнорируется (перенаправлено в nul), на работу программы не влияет
Ошибка "byte: неоднозначный символ"	Включите стандарт C++14 в настройках проекта
