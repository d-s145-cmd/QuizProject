# 🧠 Интеллектуальная викторина «Что? Где? Когда?»

Консольная игра в формате легендарной телевизионной викторины с ограничением по времени, проверкой знаний и визуальным сопровождением.

## ✨ Особенности

- ⏱️ **Таймер 30 секунд** на каждый вопрос
- 🔍 **Умная проверка ответов** (учёт синонимов и альтернативных формулировок)
- 💡 **Подробные пояснения** к каждому вопросу
- 🎯 **Подсчёт очков** и итоговая статистика
- 🌐 **Поддержка русского языка** (кодировка Windows-1251/CP866)

## 🎮 Как играть

1. На каждый вопрос даётся **30 секунд**
2. Введите ответ текстом и нажмите Enter
3. Если не успели — вопрос засчитывается как ошибка
4. После ответа (или таймаута) вы увидите:
   - Правильный ответ
   - Историческое/научное пояснение
   - Иллюстрацию (откроется автоматически)
5. Игра автоматически переходит к следующему вопросу

🔧 Технические детали
Кодировка
Вывод: Windows-1251 (SetConsoleOutputCP(1251))

Ввод: CP866 (SetConsoleCP(866))

Введённый текст автоматически конвертируется для корректного сравнения

Таймер
Реализован с использованием отдельного потока (std::thread). При истечении времени поток ввода отсоединяется (detach), чтобы не блокировать выполнение программы.

Проверка ответов
Поддерживает:разные регистры, начальные и конечные пробелы
Альтернативные формулировки (например, «да винчи» вместо «леонардо да винчи»)
Русские буквы (с учётом кодировки)
